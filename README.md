# Phone Stand Design Project

This project focuses on the design and manufacturing planning of a simple, functional phone stand suitable for desks and tables.

## Project Overview
The goal is to create a compact, lightweight phone stand that supports smartphones in both portrait and landscape orientations. The design emphasizes stability, comfortable viewing angles, and cost-effective production with minimal waste.

## Software Tools & Deliverables

### 1. CAD Modeling (SolidWorks)
Used for 3D modeling and technical drawings to ensure structural integrity and manufacturing accuracy.
* **Deliverables:** 3D model files and dimensioned technical drawings.

### 2. Process Mapping (Microsoft Visio)
Used to visualize the lifecycle of the product and user experience.
* **Deliverables:** Manufacturing process flow diagram and User interaction flow diagram.

### 3. Data Management (MySQL Workbench)
Used to design a database schema for tracking design variants, production batches, and inventory.
* **Deliverables:** ER diagram showing table relationships and SQL schema code.
